# jiraGetIssues
Get Jira Project Issue Detail Information

Usage of this script is below.
  -jiraweb string
        Jira Web Site Address, example : -jiraweb=https://inhouse.htcstudio.com/jira
  -password string
        Password
  -projectname string
        Project Name, example : -projectname=TYGH
  -username string
        User Name
